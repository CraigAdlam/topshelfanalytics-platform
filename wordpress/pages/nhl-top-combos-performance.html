Combination performance for Top Shelf Analytics’ 2+ shot combo model. This page tracks correct-combination percentage over time using completed combinations.
How to read combo performance

Each row in the combo dataset represents a 3-leg 2+ shots combination. A combination is marked correct only when every leg in the combo hits.

Combo win probability is calculated from the model probability of each leg. Expected value and edge are calculated using the combined market price.

Model-quality filters use combo-level minimum fields, meaning every player in the combination must meet the selected threshold.
